# Telegram Account Auto-Poster & Manager 🤖

Этот проект представляет собой автоматизированный инструмент для покупки, авторизации и управления Telegram-аккаунтами с использованием API торговой площадки [LOLZTEAM](https://lzt.market) и библиотеки `Telethon`.

## 🚀 Возможности

- **Автоматическая покупка** аккаунтов Telegram с LOLZTEAM по заданным фильтрам (цена, страна, происхождение)
- **Авторизация** в купленных аккаунтах через прокси (поддержка SOCKS5)
- **Настройка профиля**:
  - Случайная аватарка (из `.png` файлов в папке)
  - Случайное имя из `name.txt`
  - Случайное био из `bio.txt`
- **Настройка приватности**:
  - Открытые фото и описание
  - Запрет звонков
  - Отключение уведомлений
- **Подписка на папки с каналами** (инвайт-ссылки из `folders.txt`)
- **Автоматическое комментирование** новых постов в подписанных каналах (комментарии из `comments.txt`)
- **Мониторинг спам-статуса** через @SpamBot (каждые 10 минут)
- **Логирование** результатов работы в `Finish.txt` и `Logs.txt`

## 📁 Необходимые файлы

Перед запуском создайте следующие файлы в директории со скриптом:

- `Proxy.txt` — прокси в формате: `host:port:username:password` (по одной на строку)
- `name.txt` — имена для профиля (по одному на строку)
- `bio.txt` — описания профиля (по одному на строку)
- `comments.txt` — текст комментариев (по одному на строку)
- `folders.txt` — хеши инвайт-ссылок на папки Telegram
- `Purchased.txt` — логи покупок (создаётся автоматически)
- `Finish.txt` — финальные отчёты (создаётся автоматически)
- `Logs.txt` — статусы комментариев (создаётся автоматически)
- `id.txt` — БД уже использованных ID аккаунтов

Также поместите `.png` изображения для аватарок в ту же папку.
